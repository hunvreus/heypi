# Tools

Place Pi extension/tool files here when this example needs custom tools.

The current example also includes:

- `web_search`: public web search, using Tavily when `TAVILY_API_KEY` is configured.
- `fetch_page`: public HTTP(S) page fetch with localhost/private-IP blocking by default.

Other common capabilities still come from heypi's built-in approval, history, todo, and memory tools.
